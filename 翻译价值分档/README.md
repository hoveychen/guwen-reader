# 古籍翻译价值分档

对本仓库《殆知阁古文全集》十藏（佛/儒/史/子/集/易/道/艺/诗/医）全部古籍，按「是否值得翻译成现代白话、面向大众阅读」做的逐书分类、细分与今译核实结果。

## 数据来源

仓库根目录下 10 个藏目录（`佛藏/`、`史藏/` …）。每部书以分页 HTML 存储（`书名.html`、`书名_pageN.html`）。

- 物理 HTML 页：**279,084**
- 去分页后的真实条目（书）：**14,367**（含子分类维度计 **15,682**）

## 口径

**档位（翻译价值三档）**
- 故事性：有人物情节的叙事作品（小说/演义/话本/戏曲/笔记小说/传记/神话/佛教本生因缘故事/历史纪事）。
- 艺术文学：诗词曲赋、文集别集、文论诗话、书画音乐艺术、蒙学、楹联谜语。
- 工具经注宗教：医方、术数、方志、政书、佛道教义经咒论疏、经书注疏、考据笔记等——无翻译价值。

**今译状态**
- 确认有今译：WebFetch 实地验过，链接活且确为现代白话译注本。
- 很可能有_反爬待核：链接指向豆瓣/百科/电商等可信书目站，但被反爬拦截，无法机器抓取确认。
- 初判有_复核驳回：初判命中，二次复核时**链接打开后内容性驳回**——确为非白话学术本（梵汉对勘/许渊冲汉英对照/校注保留原文/注释本）或指向他书。注意：仅「链接失效/证书错误/404/反爬」等**抓取技术失败不再计入驳回**，已重新联网核实归位（见下）。
- 无：搜不到现代白话今译（确属翻译空白）。
- 不确定：证据模糊。
- 本身白话_无需今译：原著即白话/古白话，**现代普通读者可直接读懂、无需今译**。逐本读真实正文片段判定，非按书名朝代猜。
- 浅近文言_基本可读_待定：历史演义/平话那类半文半白，多「之乎者曰」但叙事主线现代人略费力仍能读，介于白话与文言之间，留待人工终判。
- 新增两列 `语体`/`语体依据`：仅对「章回小说/话本」逐本标注语体（白话/浅近文言/文言）及片段佐证。

## 方法

1. **逐书三档分类**：53 个 LLM agent 并行，类目先验 + 书名信号 + 古籍领域知识。较纯启发式修正约 12% 误判。
2. **题材/朝代细分**：15 个 agent 并行，受控词表判定。
3. **推荐评级**：8 个 agent 对叙事类逐书评 优先级/知名度/理由。
4. **今译核实**：32 个 agent 用 WebSearch 逐书联网核实；命中项再由 8 个 agent 用 WebFetch 逐条实地二次复核。
5. **二次复核纠偏（本轮）**：
   - 链接404纠偏：原「复核驳回」中 56 条实为抓取技术失败（证书错误/404/403/反爬），7 个 agent 重新独立联网核实，**38 本确认有今译、9 本反爬待核、5 本不确定，仅 4 本确属无**——死链 ≠ 没今译。
   - 白话原著识别：清/明/民国「章回小说+话本」469 部，13 个 agent **逐本读真实正文片段**判语体（非按书名朝代猜）——350 本判为本身白话无需今译、79 本浅近文言待定、39 本真文言需译。
   - 待核/不确定收敛：对全部「反爬待核+不确定」95 部，12 个 agent 多检索式跨站深度核实——50 本确认有今译、26 本确认无；剩余 14 本反爬待核（可信站标白话全译但被拦截）按「死链≠没今译」并入确认有今译，仅 **5 本仍不确定**（只查到校注/点校本，如述异记、云溪友议）。

## 关键数字

- 可翻译（故事+文学）≈ **4,332 部（27.6%）**；其余 72% 为工具/经注/宗教类。
- 叙事类高优先 **229** 部，其中高优先且高知名度 **85** 部（三国演义/聊斋志异/牡丹亭/窦娥冤/山海经/史记 等）。
- **翻译空白被高估，已纠偏**：原判「无今译」1,690 部中，**350 部其实是白话原著（章回/话本）本身就无需今译**，另 79 部浅近文言基本可读；原「复核驳回」56 条经重新联网，**52 条实则有今译或待核**。再对「反爬待核+不确定」95 部深度续查，**50 部确认有今译、26 部确认无**。
- **最终今译状态分布（1,955 行）**：无（翻译空白）1,291 · 本身白话_无需今译 350 · 确认有今译 212 · 浅近文言_基本可读_待定 79 · 初判有_复核驳回 18 · 不确定 5。叙事类高优先且确属翻译空白者 **72 部**（合并表左侧红条）。
- 其中「确认有今译」212 部含 14 部经可信书目站（国学典籍网/显密文库等）明标「白话全译」但被证书错误/403 反爬拦截、未能实地抓取的条目——按「死链≠没今译」一并计入，复核说明保留 `可信站标白话全译·反爬未实地核` 来源标注以便溯源。

## 局限

- 分类与评级基于书名+领域知识，存在个别误差（如「因缘」一词偶尔误判教义为故事）。
- 今译核实初判有约 1/3 假阳性，已通过二次 WebFetch 复核标注；反爬站点无法机器确认者标「待核」。**注意二次复核曾把「抓取技术失败」误当「无今译」，本轮已纠偏**——死链/证书错误/404 不再作为否定依据。
- 白话原著的语体判定基于每书一段约 1,400 字的真实正文片段，单段取样可能偶有偏差；「浅近文言_基本可读_待定」79 部为人机均不易一刀切的灰区，留待人工终判。
- **艺术文学类约 2,167 部（散文文集/诗/词/文论/书画 等）尚未纳入推荐评级与今译核实**——推荐单当前仅覆盖叙事/故事类九个题材。
