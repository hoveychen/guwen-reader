# 中华古籍阅读器 · GuWen Reader

一个**纯静态、零依赖**的现代古籍阅读器，覆盖佛 / 儒 / 史 / 子 / 集 / 易 / 道 / 艺 / 诗 / 医**十藏共 15,694 部**已整理为 Markdown 的古籍，并附带《古籍翻译价值分档》数据集。

**🌐 在线阅读：<https://hoveychen.github.io/guwen-reader/>**（GitHub Pages 仅部署阅读器与搜索索引，正文按需从本仓库 raw 加载）

也可本地用浏览器打开 `index.html` 阅读（需经 HTTP 服务，见下）。

## 功能

- **即时搜索**：按书名 / 作者 / 分类实时过滤 15,694 部，命中高亮，点击直接展卷。
- **书目树**：十大藏 → 子分类 → 书，懒加载；落地页十藏卡片导航。
- **自动章节目录**：对纯文本正文启发式识别卷 / 篇 / 论等小标题，并收敛重复的分隔标签。
- **护眼排版**：宋体米黄纸背、阅读进度条、返回顶部。
- **横 / 竖排一键切换**、字号增减、日 / 夜间模式。
- **记忆上次位置**、URL `#` 路由可分享、移动端适配、键盘翻页。

## 本地运行

仓库较大且阅读器通过 `fetch` 按需加载各书 Markdown，**必须经 HTTP 服务访问**，直接 `file://` 打开会因跨源限制失败：

```bash
cd guwen-reader
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000/
```

## 目录结构

| 路径 | 说明 |
|---|---|
| `index.html` | 阅读器（单文件，原生 JS，无框架） |
| `manifest.json` | 全量书目索引（15,694 部，约 1.8MB），供前端搜索 / 浏览 |
| `markdown/` | 十藏全部古籍正文，`<藏>/[子分类/]书名.md` |
| `翻译价值分档/` | 对全部古籍「是否值得翻译成现代白话」的逐书分档、题材朝代细分、今译核实数据集（CSV + 自包含网页），详见其内 `README.md` |
| `tools/` | 数据处理脚本：`build_manifest.py`（重生书目索引）、`html_to_md.py`、`build_merged_html.py` |

### 重建书目索引

新增 / 删改 `markdown/` 下的书后，重新生成 `manifest.json`：

```bash
python3 tools/build_manifest.py
```
